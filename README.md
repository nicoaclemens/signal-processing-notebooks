## How to use

```bash
git clone https://github.com/NicoaClemens/Signal-Processing-Notebooks.git
cd Signal-Processing-Notebooks
python -m venv venv
# win
.\venv\Scripts\activate
.\run
# linux macos
source venv/bin/activate
./run.sh
```
